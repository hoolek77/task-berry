## Usage

```bash
yarn install # installs dependecies
yarn start # compiles and hot-reloads for development
yarn build # compiles and minifies for production
yarn lint # lints files
yarn lint:fix # lints and fixes files
```

## Tech Stack
- React + TypeScript
- Redux Toolkit
- StyledComponents
- Gsap
- i18n
- Axios
